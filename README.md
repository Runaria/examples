# Examples

Project examples for SPM.

## Prepare

```bash
$ npm install spm -g
$ npm install spm-server -g
```

## Usage

e.g. todo

```bash
## enter directory
$ cd todo

## install dependencies
$ spm install
$ open http://127.0.0.1:8000/

## debug
$ spm-server

## build
## install node dependencies if have
$ npm install
$ spm build
$ open dist/index.html
```
