# Project Examples for SPM

## Install

```bash
$ npm install spm -g
$ npm install spm-server -g
```

`spm-server` is for debug.

## Usage

```bash
$ cd todo
```

install dependencies

```
$ spm install
```

debug

```bash
$ spm-server
```

build

```bash
$ spm build
```

Visit [spm-handbook](http://sorrycc.gitbooks.io/spm-handbook/content/) for more infomation.
